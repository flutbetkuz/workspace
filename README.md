# AI First Workspace

Централизованное управление проектами и документацией.

## Структура

```
workspace/
├── Strategy/      - Стратегическое планирование, бизнес-модель
├── Docs/          - Документация проектов и процессов
├── Marketing/     - Маркетинговые материалы и кампании
├── Operations/    - Операционные процессы, найм, метрики
├── Projects/      - Информация о проектах (код в отдельных репо)
└── scripts/       - Утилиты и автоматизация
```

## Использование

Этот workspace предназначен для хранения документации, стратегий и информации о проектах. Код проектов хранится в отдельных репозиториях.

### Multi-root Workspace

Для работы с несколькими репозиториями одновременно используйте файл `my-workspace.code-workspace` в родительской директории.
