# Paradox Therapy Bot

Telegram-бот для рефлексии без давления и геймификации.

## Репозиторий

**GitHub:** https://github.com/flutbetkuz/paradox-therapy-bot
**VPS:** Запущен в production (Docker)

## Описание

Бот для людей, уставших от культуры продуктивности. Задаёт вопросы пачками по 3 штуки, хранит полную приватность пользователя, собирает анонимные следы в ежемесячный дайджест.

## Ключевые принципы

- **Минимализм** — без streak'ов, напоминаний, давления
- **Полная приватность** — ответы НЕ сохраняются на сервере
- **Честный тон** — без эмодзи, восклицательных знаков
- **Антигеймификация** — не превращаем осознанность в KPI

## Технологии

- Python 3.10+, aiogram 3.x
- SQLite, APScheduler
- Docker (production), OpenRouter API (дайджест)

## Команды бота

### Публичные
- `/start` — приветствие и онбординг
- `/paradox` — следующий вопрос из пачки (1/3 → 2/3 → 3/3)
- `/digest` — ежемесячная сводка сообщества
- `/subscribe` — подписка ($2/мес)
- `/about` — о приватности
- `/archive` — архив старых сводок (для подписчиков)

### Скрытые (admin)
- `/grant_premium` — выдать премиум пользователю
- `/gen_digest` — принудительно сгенерировать сводку
- `/stats` — статистика пользователей
- `/broadcast` — рассылка всем пользователям

## Механика

- **Пачка вопросов:** 3 вопроса подряд без ожидания
- **Интервал:** free — 5 дней, premium — 3 дня (после 3/3)
- **Следы:** любое сообщение → предложение поделиться анонимно
- **Дайджест:** раз в месяц, генерируется через OpenRouter LLM
- **Подписка:** код готов, Telegram Payments не подключены
