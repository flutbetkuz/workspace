## Структура

Каждый проект имеет свою папку с:
- README.md - Основная информация
- Ссылка на репозиторий
- Статус разработки
- Связанная документация

## Добавление нового проекта

1. Создать папку с названием проекта
2. Добавить README.md с информацией
3. Создать отдельный репозиторий для кода
4. Обновить этот файл
